Paso 1: Descarga el archivo base.

Lo primero que debes hacer es descargar nuestra plantilla de ejemplo para empezar a crear tu propio objeto de aprendizaje. Haz click en el botón "Descargar Plantilla".

Paso 2: Escribiendo con etiquetas.

Una vez descargado el archivo base, debes escribir todo el contenido de tu objeto de aprendizaje usando las siguientes etiquetas, dependiendo las funcionalidades que deseas agregar. Para ver las etiquetas haz click en en link "Mostrar etiquetas".

Mostrar etiquetas

Ocultar etiquetas
Etiqueta / Fin etiqueta Descripción
<objeto titulo="tituloObjeto" autor="autorObjeto" tema="temaObjeto"> </objeto> Una vez definida la etiqueta <comenzar>, para darle un título, autor y tema al objeto de aprendizaje se deben agregar los atributos titulo, autor, y tema.
<escena titulo="nombreEscena" tipo="tipoEscena"> </escena> Etiqueta que indica el inicio y final de una escena, indicando el nombre de la escena para su identificación y un tipo, todo lo que se defina dentro de estas etiquetas estará contenido en la escena del objeto de aprendizaje.
<bloque > </bloque> Etiqueta que define el bloque en particular del objeto de aprendizaje. Depende de la cantidad de filas y/o columnas que se definan.
<idea orden="numeroAparicion"> </idea> Etiqueta que define la idea que aparecerá en el bloque en particular del objeto de aprendizaje. El atributo orden indica el orden de aparición de esa idea.
<texto tipo="tipoTexto"> </texto> Etiqueta que define el texto que aparecerá en la idea en particular del objeto de aprendizaje. Los tipos pueden ser; normal, manuscrito y código.
<media tipo="imagen"> </media> Etiqueta que contiene la URL desde donde se descargará una imagen desde Internet. El tipo indica que es una imagen.
<ejemplos> </ejemplos> Etiqueta que define la pila de ejemplos que se quiera mostrar.
<ejemplo> </ejemplo> Etiqueta que define un ejemplo a mostrar.
<texto_ejemplo tipo="normal"> </texto_ejemplo> Etiqueta que define el texto que tendra un ejemplo en particular.
<media_ejemplo tipo="imagen"> </media_ejemplo> Etiqueta que define una imagen que se mostrará en un ejemplo en particular.
<voz> </voz> Etiqueta que contiene un texto que será leido por una voz en tiempo real en la idea en particular del objeto de aprendizaje.
<evaluacion exigencia="cantidadExigencia"> </evaluacion> Etiqueta que define una evaluación, la cual contendrá preguntas a evaluar. La exigencia va desde 1 a 100.
<pregunta> </pregunta> Etiqueta que define la cantidad de preguntas que tiene una evaluación.
<forma> </forma> Etiqueta que define la pregunta que contiene una evaluación. Debe contener una etiqueta <enunciado>, <opciones> y <solucion>
<enunciado> </enunciado> Etiqueta que define el enunciado que tendrá la forma. Debe contener una etiqueta <texto>
<opciones> </opciones> Etiqueta que define la cantidad de opciones que tiene la forma. Debe contener una etiqueta <alternativa>
<alternativa tipo="tipoAlternativa" tema="nombreTema"> </alternativa> Etiqueta que define las alternativas que puede tener una forma, su tipo puede ser "solucion" o "distractor" y el tema es el nombre del tema de la alternativa. Debe contener una etiqueta <texto>
<solucion> </solucion> Etiqueta que define la solución que se va a mostrar por cada forma. Debe contener una etiqueta <texto>
<feedback> </feedback> Etiqueta que define el feedback en forma de encuesta.

Paso 3: Genera tu OA!

Cuando hayas terminado de crear tu archivo XML con las respectivas etiquetas, ve a la página de inicio para generarlo, presiona el botón "Genera tu objeto de aprendizaje Ahora!" y cárgalo en la barra. Luego presiona el botón "Generar" y te redireccionará automaticamente para visualizarlo al instante.
