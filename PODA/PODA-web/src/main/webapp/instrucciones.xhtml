Paso 1: Descarga el archivo base!

Lo primero que debes hacer es descargar nuestra plantilla de ejemplo para empezar a crear tu propio objeto de aprendizaje. Haz click en Descargar Plantilla

Paso 2: Escribiendo con etiquetas!

Una vez descargado el archivo base, debes escribir todo el contenido de tu objeto de aprendizaje usando las siguientes etiquetas dependiendo las funcionalidades que deseas agregar. Para ver las etiquetas haz click en en link Mostrar etiquetas.

Mostrar etiquetas

Ocultar etiquetas
Etiqueta / Fin etiqueta Descripción
<object> </object> Etiqueta para indicar el inicio y final del objeto de aprendizaje, todo lo que se encuentre dentro de esta etiqueta corresponderá al objeto de aprendizaje.
<object title="tituloObjeto"> </object> Una vez definida la etiqueta anterior, para darle un título al objeto de aprendizaje la etiqueta anterior se complementa con title.
<scene sceneTitle="nombreEscena"> </scene> Etiqueta que indica el inicio y final de una escena, indicando el nombre de la escena para su identificación, todo lo que se defina dentro de estas etiquetas estará contenido en la escena del objeto de aprendizaje.
<text type="text"> </text> Etiqueta que define el texto que aparecerá en la escena en particular del objeto de aprendizaje.
<voice> </voice> Etiqueta que contiene un texto que será leido por una voz en tiempo real en la escena en particular del objeto de aprendizaje.

Paso 3: Genera tu OA!.

Cuando hayas terminado crear tu archivo XML, cárgalo en la barra de arriba, presiona el bóton "Generar", luego presiona el bóton "Ir a OA" y verás tu objeto de aprendizaje ya generado.
