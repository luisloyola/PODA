Historia

El proyecto PODA nace como una propuesta hacia lo que es la creación de objetos de aprendizaje, de manera alternativa a los que es el uso de PowerPoint, Flash y otras herramientas, permitiendo crear estos objetos de aprendizaje de una forma estándar obteniendo resultados visualmente coloridos, estilizados y modernos, con funcionalidades especiales como voz en tiempo real, ejemplos y evaluaciones aleatorias, uso de recursos multimedias y animaciones, todo esto sin dejar de lado la simplicidad y la amigabilidad con el usuario.
