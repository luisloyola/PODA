Paso 1: Descarga el archivo base!

Lo primero que debes hacer es descargar el archivo base para empezar a crear tu objeto de aprendizaje, este lo puedes descargar desde Aquí

Paso 2: Escribiendo con etiquetas!

Una vez descargado el archivo base, debes colocar algunas de las etiquetas dependiendo de lo que quieras visualizar en tu objeto de aprendizaje, como imágenes, texto, audio, etc. Para ver las etiquetas haz click en en link Mostrar etiquetas.

Mostrar etiquetas

Ocultar etiquetas

Etiqueta / Fin etiqueta Descripción
<object title="tituloObjeto"> </object> Etiqueta principal que indica la intención de crear un objeto de aprendizaje
<scene sceneTitle="nombreEscena"> </scene> Define el dominio de una escena, aquellos elementos que se mostrarán juntos en el flujo de reproducción
<text type="text"> </text> Define el espacio en donde se encuentra texto que se desea mostrar en cierta escena
<voice> </voice> Define el espacio del texto que debe ser incluido para ser reproducido

Paso 3: Genera tu OA!.

Cuando hayas terminado crear tu archivo XML, cárgalo en la barra de arriba, presiona el bóton "Generar", luego presiona el bóton "Ir a OA" y verás tu objeto de aprendizaje ya generado.
