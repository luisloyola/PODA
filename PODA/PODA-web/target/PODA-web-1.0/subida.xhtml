Paso 1: Descarga el archivo base.

Lo primero que debes hacer es cargar el archivo base para empezar a crear tu Objeto de Aprendizaje, este lo puedes descargar desde Aquí

Paso 2: Colocándo etiquetas.

Una vez descargado el archivo base, debes colocar algunas de las equiquetas dependiendo de lo que quieras visualizar en tu Objeto de Aprendizaje, como imágenes, texto, audio, etc. A continuación, está el listado de etiquetas que puedes utilizar:

Etiqueta Descripción
scene Esta etiqueta indica el slide de la presentación
Title Esta etiqueta indica el título
audio Esta etiqueta indica que ese párrafo debe ser hablado

Paso 3: Guardando archivo.

Cuando hayas terminado de generar tu Objeto de Aprendizaje, guárdalo en tu computador, búscalo y estarás listo para generar tu objeto de aprendizaje.
